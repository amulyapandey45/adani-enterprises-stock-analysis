# Adani Enterprises Stock Data Analysis Dashboard

An interactive **Power BI dashboard** built to analyze the stock performance of **Adani Enterprises Ltd.** using key financial metrics like **Open, Close, Volume, 52 Week High, and 52 Week Low**.

The dashboard provides **daily, monthly, quarterly, and yearly analysis** with dynamic filters, KPI cards, and visual reports to track stock trends and market performance effectively.

---

## Features

* Interactive Power BI Dashboard
* KPI Cards for stock insights
* Daily, Monthly & Quarterly trend analysis
* Volume analysis by month
* Open vs Close price comparison
* 52 Week High & Low tracking
* Dynamic slicers for Day, Month, Quarter & Year

---

## Tools Used

* Power BI
* Excel / CSV Dataset
* DAX
* Data Visualization

---

## Dashboard Insights

* Stock movement analysis
* Trading volume trends
* Comparative price analysis
* Quarterly performance overview
