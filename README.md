# Adani Enterprises Stock Analysis
Stock market analysis project of Adani Enterprises using Excel and Power BI. This project explores historical price trends, trading volume, returns, and volatility through interactive dashboard visualizations.

## Tools Used
- Excel  
- Power BI  
- Financial Data Analysis  

## Key Features
- Price Trend Analysis  
- Volume Analysis  
- Return & Volatility Insights  
- Interactive Dashboard  

## Files
- Historical Stock Dataset (.xlsx)  
- Power BI Dashboard (.pbix)
